# MOOSTIK - Chronologie Narrative

## Vue d'Ensemble

L'histoire de MOOSTIK se divise en trois ères distinctes, chacune avec son atmosphère visuelle et narrative propre.

```
┌─────────────────────────────────────────────────────────────────┐
│                    TIMELINE MOOSTIK EP0                          │
├──────────────────┬──────────────────┬──────────────────────────┤
│  AVANT GÉNOCIDE  │ PENDANT GÉNOCIDE │    APRÈS GÉNOCIDE        │
│    (Paix)        │   (Apocalypse)   │    (Reconstruction)      │
│                  │                  │                          │
│  Village idyllique│  Spray BYSS     │  Bar Ti Sang (présent)   │
│  Vie quotidienne │  Destruction     │  Récits des survivants   │
│  Innocence       │  Sacrifice       │  Naissance Bloodwings    │
└──────────────────┴──────────────────┴──────────────────────────┘
```

## Ère 1 : Avant le Génocide

### Timeline Position

```typescript
timeline_position: "before_genocide"
```

### Contexte

Le village **Cooltik** vit dans une paix relative. Les Moostiks mènent leur vie quotidienne sans se douter de la catastrophe imminente.

### Atmosphère Visuelle

```json
{
  "lighting": "Golden hour, warm and idyllic",
  "atmosphere": ["warm humidity", "soft pollen particles", "gentle dew mist"],
  "mood": "Peaceful tranquility, innocence, daily joy",
  "color_grade": "Warm, saturated, hopeful"
}
```

### Événements

| Shot | Description |
|------|-------------|
| Village paisible | Vie quotidienne du village Cooltik |
| Activités communautaires | Moostiks en interaction sociale |
| Derniers moments heureux | Avant l'ombre de l'enfant |

### Personnages Présents

- Mama Dorval et Baby Dorval
- Villageois ordinaires
- Pas encore de Bloodwings

## Ère 2 : Pendant le Génocide

### Timeline Position

```typescript
timeline_position: "during_genocide"
```

### Contexte

L'enfant caribéen, jouant innocemment, déclenche le spray BYSS qui détruit le village et tue la majorité des Moostiks.

### Atmosphère Visuelle

```json
{
  "lighting": "Violent crimson glow, toxic haze",
  "atmosphere": ["toxic BYSS aerosol fog", "ember particles", "soot smoke", "ash flecks"],
  "mood": "Apocalyptic horror, chaos, mass death",
  "color_grade": "Desaturated with crimson highlights, heavy contrast"
}
```

### Événements Clés

| Shot | Description | Personnages |
|------|-------------|-------------|
| L'Ombre de la Mort | Main de l'enfant approche | L'Enfant (mains) |
| Le Sacrifice de Maeli | Mama protège Baby | Mama Dorval, Baby Dorval |
| Le Nuage Toxique | Spray BYSS se répand | Villageois |
| Les Premières Victimes | Moostiks touchés par le spray | Victimes |
| Le Dernier Souffle | Mama Dorval meurt | Mama Dorval |
| L'Éveil du Survivant | Baby Dorval seul | Baby Dorval |

### Le Sacrifice de Mama Dorval

```
Mama Dorval sacrifie sa vie pour sauver son enfant.
Elle l'enveloppe de ses ailes, le protégeant du spray toxique.
C'est le moment central de l'épisode - le sacrifice ultime d'une mère.
```

## Structure Narrative EP0

### Framing Device

L'épisode utilise un **cadre narratif** :
- **PRÉSENT** : Papy Tik au Bar Ti Sang raconte l'histoire
- **FLASHBACK** : Les événements du génocide
- **RETOUR** : Réaction émotionnelle des auditeurs

### Séquence des Shots

```
┌─────────────────────────────────────────────────────────────────┐
│                        EP0 STRUCTURE                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ACTE 1: Introduction au Bar                                    │
│  ├── Shot 11: Voix du Passé (présent)                          │
│  └── Transition vers flashback                                  │
│                                                                 │
│  ACTE 2: Le Génocide (flashback)                               │
```
